# Flight Deck visual assets

`earthrise.webp` was generated for RFC-004.1 with OpenAI's built-in image
generation tool on 2026-07-21. It is an original, project-local asset: no
third-party image or runtime request is involved.

Production prompt summary: a scientifically plausible, photorealistic orbital
Earthrise with the night side, restrained city lights, a thin atmospheric limb,
and generous dark negative space for the Flight Plan overlay. The prompt
explicitly excluded UI, text, logos, spacecraft, people, fantasy elements, and
watermarks.

The selected 1774×887 PNG was converted to an 86-quality WebP. The production
asset is approximately 112 KB.
